# Portfolio | Lukas Penndorf 🚀

Willkommen im GitHub-Repository für mein digitales Portfolio! 

Dieses Repository enthält den Quellcode für meine persönliche Webseite, auf der ich meine Arbeiten und Projekte aus dem Studiengang **Digital Media Marketing** an der Hochschule Kaiserslautern (University of Applied Sciences) präsentiere.

## 🎯 Über das Portfolio
Mein Fokus liegt auf der Schnittstelle zwischen kreativem Design, technischer Umsetzung und strategischem Marketing. Das Portfolio dient als digitale Projektmappe für meine akademischen und praktischen Arbeiten und demonstriert einen effizienten digitalen Workflow.

## 📂 Ausgewählte Projekte in diesem Repository
* **Web-Entwicklung & SEO:** Performante One-Pager (z.B. fiktives Café Zweibrücken) mit Fokus auf semantischem HTML5, strukturierten Daten (Schema.org) und technischer Suchmaschinenoptimierung.
* **Wissenschaftliche Methodik & Transfer:** Akademische Aufarbeitung neurobiologischer Suchtmechanismen in Social Media und deren journalistischer Transfer in ein professionelles Magazin-Layout ("ZEIT Wissen" Stil).
* **Graphische Gestaltung & Digitaler Workflow:** Entwicklung der Interior-Marke „RAUMLUFT". Hier zeige ich das Zusammenspiel von KI-generiertem Prototyping (Google Gemini / Adobe Firefly) und professionellem manuellen Layout-Handwerk (Affinity Suite).

## 🛠️ Tech Stack & Tools
* **Web:** HTML5, CSS3, Vanilla JavaScript, Responsive Design
* **Design & Layout:** Affinity Suite
* **KI & Prototyping:** Google Gemini
* **Workflow:** Git, GitHub Pages

# English version | Lukas Penndorf 🚀

Welcome to the GitHub repository for my digital portfolio! 

This repository contains the source code for my personal website, where I showcase my work and projects from the **Digital Media Marketing** program at the University of Applied Sciences Kaiserslautern.

## 🎯 About the Portfolio
My focus lies at the intersection of creative design, technical implementation, and strategic marketing. The portfolio serves as a digital project portfolio for my academic and practical work and demonstrates an efficient digital workflow.

## 📂 Selected Projects in This Repository
* **Web Development & SEO:** High-performance one-pagers (e.g., fictional Café Zweibrücken) with a focus on semantic HTML5, structured data (Schema.org), and technical search engine optimization.
* **Scientific Methodology & Transfer:** Academic analysis of neurobiological addiction mechanisms in social media and their journalistic adaptation into a professional magazine layout (in the style of “ZEIT Wissen”).
* **Graphic Design & Digital Workflow:** Development of the interior design brand “RAUMLUFT.” Here I demonstrate the interplay between AI-generated prototyping (Google Gemini / Adobe Firefly) and professional manual layout craftsmanship (Affinity Suite).

## 🛠️ Tech Stack & Tools
* **Web:** HTML5, CSS3, Vanilla JavaScript, Responsive Design
* **Design & Layout:** Affinity Suite
* **AI & Prototyping:** Google Gemini
* **Workflow:** Git, GitHub Pages
